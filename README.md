
# Auth-SpringSecurity

## Descripción

Este proyecto es una implementación de autenticación y autorización utilizando Spring Boot y Spring Security.  
Está diseñado para proporcionar una base sólida en la gestión de usuarios y roles, integrando prácticas de seguridad modernas como JWT (JSON Web Tokens) y autenticación básica.

## Características

- **Autenticación básica**: Implementación de autenticación básica para proteger los endpoints.
- **JWT (JSON Web Token)**: Generación y validación de tokens JWT para sesiones seguras.
- **Roles y permisos**: Definición de roles y permisos para controlar el acceso a recursos específicos.
- **Integración con JPA/Hibernate**: Persistencia de datos utilizando JPA y Hibernate.
- **Uso de Lombok**: Reducción de código boilerplate con la biblioteca Lombok.

## Tecnologías Utilizadas

- **Backend**:
  - Java 17
  - Spring Boot 2.5.x
  - Spring Security
  - Spring Data JPA
  - JWT (JSON Web Tokens)
  - Lombok
  - H2 Database (para pruebas)

## Requisitos

- Java 17 o superior
- Maven
- IDE recomendado: IntelliJ IDEA o Eclipse
